Trading Functions

Trading functions are a specific way of doing automated market making. Given a set of reserves and a single invariant, we can know how to compute a swap between any pair of assets. Trading functions are completely deterministic e.g. in their price discovery properties.

Quantities we can compute:

  • Price
  • Price impact
    • Volume to volatility at a given price
  • Liquidity

Notation

  • The tokens are given by: X≡token_x Y≡token_y≡
  • The reserves are: x≡reserve_x y≡reserve_y
  • The trading function is: φ(x,y)≡trading_function
  • The fee parameter is: γ≡fee
  • The amount changed will be written as δ with subscripts and will be given by: δ≡amount_deposit δ≡amount_withdrawn we can use a sign on δ to denote whether it is a deposit or withdrawal.
    • For liquidity deposits: δx​≡amount_deposit_x δy​≡amount_deposit_y
    • For liquidity withdraws: δx​≡amount_withdrawn_x δy​≡amount_withdrawn_y
  • The amount swapped is written as Δ
  • The pool price is: p≡pool_price
  • The price is: S≡price This can show up for arbitrage math for example

Computations

Pool Price

The pool price p can be computed by: p=∇y​ψ∇x​ψ​ This assumes that Y is the numeraire token, so that this is a price of X in terms of Y.

Liquidity Changes

When finding a valid change in liquidity, we must assert that the pool price p remain invariant. That is: p(x,y)=p(x+δx​,y+δy​)

Swaps

When finding a valid (fee-less) swap, we must assert that the trading function remain invariant. That is φ(x,y)=φ(x+Δx​,y+Δy​) This is, assuming there is no fee.

Including a Reinvested Fee

If we include a fee, then we must assert that the trading function remain invariant after the fee is taken and reinvested into the pool. The logic works this way, but when it comes to defining the equations, we can do so without having to take multiple steps. In essence:

  • The user inputs a Δ and (1−γ)Δ is taken as a liquidity change. This increases the liquidity first, and then the swap is done by taking the remaining γΔ and using the equation of a swap without a fee.

So, we have, for a swap taking in X: φ(x+Δx​,y+Δy​)=φ(x,y)⟹Δy​. Then we must deposit δx​=(1−γ)Δx​ and from which it must be that: p(x+δx​,x+δy​)=p(x,y)⟹δy​. The user then receives back: Δx​​=Δy​−δy​. The new invariant value for the trading function is then: φ(x+δx​,y+δy​).
